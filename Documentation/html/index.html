MAXimus

This library is very flexible and can be easily altered by it's users if they so desire.


Examples of it's usability

  • Simple animations (using multiple matrices)
  • Mathematical shapes (squares, circles, lines, triangles)
  • String to text output
  • The ability to scale the display size easily

    • Example of some c++ code

    #include "hwlib.hpp"
    #include "MAXimus.hpp"
    int main(void){
     // kill the watchdog
     WDT->WDT_MR = WDT_MR_WDDIS;
     auto cs = hwlib::target::pin_out(hwlib::target::pins::d12);
     auto clk = hwlib::target::pin_out(hwlib::target::pins::d11);
     auto din = hwlib::target::pin_out(hwlib::target::pins::d13);
     hwlib::wait_ms(1000);
     display w( cs, clk, din);
     shapes s(w);
     w.init();
     w.setBrightness(70);
     w.clean();
     s.line(coordinate(1,1), coordinate(20,8)
     }
