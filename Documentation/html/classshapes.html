EXTREMELY fast line algorithm made by a Professor at MIT.

This function uses a heavily modified version of the xiaolin wu anti-aliasing algorithm.
This modified version removes the contrast correction that comes standard to the original version and enhances some other features like less redundant code.
This algortihm also is very optimized for smaller microcontrollers or hardware in general by using bitshifting instead of diferncias in more loops.
