\subsection{Detailed Description}
A simple display driver that uses a axessystem with an inverted y-\/axis

This class controls all the operations used to write and delete items on the led display. ~\newline
A custom 2d matrix array is used to store temporary data which will be written after each update. ~\newline
\begin{DoxyWarning}{Warning}
Only the y-\/axis on the D\+I\+S\+P\+L\+AY is inverted. To draw pixels etc you can use a normal (x,y) axis system. 
\end{DoxyWarning}
